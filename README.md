User Management Dashboard

A responsive User Management Application built with React.js. This application allows users to perform CRUD (Create, Read, Update, Delete) operations by interacting with the JSONPlaceholder API.

Live Demo: [https://usermanagementdashboard-assignment-6.netlify.app/]

Features

View Users: Fetches and displays a list of users from an external API.
Add User: Allows adding new users via a form with validation.
Edit User: Functionality to update existing user details.
Delete User: Remove users from the list with a confirmation prompt.
Error Handling: Displays error messages for failed API requests or invalid inputs.
Responsive UI: Clean and professional interface using pure CSS.

Tech Stack

Frontend: React.js (Hooks: useState, useEffect)
HTTP Client: Axios (for API requests)
Styling: CSS3 (Flexbox/Grid)
API: JSONPlaceholder (Fake REST API)
